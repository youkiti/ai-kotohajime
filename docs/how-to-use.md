# 使い方

## 準備するもの

- 手元の生成AI: ChatGPT、Claude、Gemini、Copilotなど。無料版で構いません。有料登録は不要です。
- ブラウザ: このサイトを開くブラウザだけです。Gitやソフトウェアのインストールは必要ありません。
- 所要時間の目安: 1ユニットあたり10分程度、領域クイズは5〜10分程度です。

## 学習の5ステップ

1. **ユニットを読む** — 各領域のユニットページで、なぜそのスキルが必要か、コアの解説、プロンプト例を確認します。
2. **プロンプトを試す** — ユニットに掲載されているコピー可能なプロンプトを、手元の生成AIに貼り付けて実行します。
3. **フィードバックAIに提出する** — 出てきた結果と使ったプロンプトを[フィードバックAI](feedback.md)に提出し、講評を受けます。
4. **領域クイズに挑戦する** — その領域のユニットを終えたら、領域クイズ(10問・四択)に挑戦します。8割(8問)以上の正解で合格です。不正解の設問には解説が表示されます。
5. **目録を発行する** — クイズに合格すると「目録を発行」ボタンが表示されます。氏名(ニックネーム可)を入力してPNG画像を生成し、ダウンロードやSNSでの共有ができます。

!!! tip "すでに使い慣れている方へ"
    生成AIを普段から使っている方は、ユニット本文を読まずに領域クイズへ直行しても構いません。**目録の発行条件はクイズ合格のみ**で、ユニットの閲覧は問いません。

## よくある質問

??? question "費用はかかりますか?"
    このサイト自体は無料です。手元の生成AIも、無料版の範囲で十分に体験できるよう内容を設計しています。有料プランへの登録は必要ありません。

??? question "アカウント登録は必要ですか?"
    不要です。このサイトにログインの仕組みはありません。

??? question "学習の記録はどこに保存されますか?"
    クイズの合格状態は、あなたが使っているブラウザの中(localStorage)にのみ保存されます。サイト側のサーバーには一切送信されません。ブラウザを変えたり、履歴・サイトデータを消去したりすると記録は失われます。詳しくは[プライバシー方針](privacy.md)をご覧ください。
