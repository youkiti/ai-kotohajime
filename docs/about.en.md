# About This Site

## Origin of the Name

The name "AI Kotohajime" (AI事始) draws on Sugita Genpaku's *Rangaku Kotohajime* (The Beginnings of Dutch Learning), written by the Edo-period physician of Dutch medicine. With almost no teacher to guide them, the scholars of that era worked their way, largely by trial and error, through *Tāheru Anatomia*, a Dutch-language anatomy text. We see a parallel between those pioneers and today's healthcare professionals, who are likewise teaching themselves to work with the new tool of generative AI.

## Design Principles

This site is built on the following four principles.

1. **Do not add to teachers' workload.** A teacher's job is only to send a link and say, "Take a look at this."
2. Learners can complete the entire journey — the experience itself, the feedback, and issuing a certificate — on their own, without anyone's help.
3. A single administrator can maintain the site by editing only Markdown. The authoritative source for the AI instruction prompts is likewise Markdown, managed within this repository.
4. **The site retains no user logs or personal information whatsoever.** (Saving quiz-passing status to localStorage takes place entirely on the learner's own device, so it does not conflict with this principle.)
