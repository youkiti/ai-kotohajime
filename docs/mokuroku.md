# 目録(修了証)

「目録」は、領域クイズに合格すると自分で発行できる修了証です。検証機構を持たない自己申告型のツールであり、**公式な研修証憑ではありません**。氏名欄はニックネームでも構いません。入力した氏名はブラウザ内の画像生成にのみ使われ、どこにも送信されません。生成した画像はダウンロードやSNSでの共有ができます。
