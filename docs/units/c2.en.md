# Unit C-2 Copyright, Bias, and Verification: Etiquette Before You Use It

**Corresponding Competencies**: C2, C3, C4 | **Estimated Time**: 10 minutes (planned)

!!! warning "Under preparation (mock)"
    This unit is currently only a skeleton placeholder. Content is planned to be added, in a structure along these lines:

    - Why this skill matters
    - Core explanation (copyright and ethical considerations around generated output and training data, bias arising from training data, and the need for human verification before using output in your work)
    - Copyable prompts (e.g., turning verification points into a checklist)
    - Steps
    - Example of expected output
    - Common failure patterns and how to address them
    - Safety notes
    - Self-check quiz (knowledge questions plus case judgment, scenario format: what part of this output should be verified?)
    - Feedback on your work
