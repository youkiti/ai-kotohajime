# AI事始(AIことはじめ)

**医療者のための生成AI自習サイト**

!!! warning "モック版です(A領域のみ実動)"
    現在公開しているのは、サイト構成を確認するためのモック版です。実際に学習してクイズに合格し、目録(修了証)を発行できるのは**領域A: 生成AIのメカニズム**のみです。領域B・Cは画面の骨格のみ用意しており、内容は準備中です。

## 「ここを見ておいて」と言われた方へ

指導医や教員から本サイトのリンクを送られた方は、このページの案内に沿って進めてください。登録もアカウント作成も不要です。普段お使いのブラウザと、手元の生成AI(ChatGPT、Claude、Gemini、Copilotなど)があればすぐに始められます。

## 進め方は3ステップ

1. **学ぶ** — 各ユニットを読み、掲載されているプロンプトを手元の生成AIに貼って試します。1ユニット10分程度です。
2. **クイズに挑戦する** — 領域ごとのクイズ(10問・四択)に挑戦します。8割正解で合格、何度でも挑戦できます。
3. **目録を発行する** — クイズに合格すると、その場で「目録」(修了証)を発行できます。氏名はニックネームでも構いません。

詳しい手順は[使い方](how-to-use.md)をご覧ください。

## 3つの領域

- **領域A: 生成AIのメカニズム** — LLMや画像生成の仕組み、ハルシネーションが起こる理由を学びます。✅ モック版で体験できます。[領域Aへ →](units/a1.md)
- **領域B: 対話的活用スキル** — 反復改善、コンテキストの与え方、AIに質問・批評させる技法を学びます。🚧 準備中です。[領域Bへ →](units/b1.md)
- **領域C: 倫理・安全・検証** — 患者情報の保護、著作権、バイアス、人による検証を学びます。🚧 準備中です。[領域Cへ →](units/c1.md)

## このサイトの約束

- **登録不要** — アカウント作成もログインも必要ありません。
- **ログ収集ゼロ** — アクセス解析を含め、利用状況はサイト側に一切記録されません。詳しくは[プライバシー方針](privacy.md)をご覧ください。
- **目録は公式な証明ではありません** — 学習の達成感と、任意の修了報告のための自己発行ツールです。詳しくは[免責事項](disclaimer.md)をご覧ください。
- **教員の仕事はリンクを送るだけ** — 詳しくは[教員の方へ](for-teachers.md)をご覧ください。
