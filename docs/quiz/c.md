# 領域Cクイズと目録発行

## ルール

- 全10問・四択形式です。
- 全問に回答してから、まとめて採点します(一括採点)。
- 8問(8割)以上の正解で合格です。
- 不正解だった設問には解説が表示されます。
- 何度でも再挑戦できます。回数制限はありません。
- [Unit C-1](../units/c1.md)・[Unit C-2](../units/c2.md)を読んでいなくても挑戦できます。合格の条件はクイズの正解のみです。
- 合格記録はこの端末のブラウザ内(localStorage)にのみ保存されます。サーバーには送信されません。
