# 領域Aクイズと目録発行

## ルール

- 全10問・四択形式です。
- 全問に回答してから、まとめて採点します(一括採点)。
- 8問(8割)以上の正解で合格です。
- 不正解だった設問には解説が表示されます。
- 何度でも再挑戦できます。回数制限はありません。
- [Unit A-1](../units/a1.md)・[Unit A-2](../units/a2.md)を読んでいなくても挑戦できます。合格の条件はクイズの正解のみです。
- 合格記録はこの端末のブラウザ内(localStorage)にのみ保存されます。サーバーには送信されません。
