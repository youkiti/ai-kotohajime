# このサイトについて

## 名称の由来

「AI事始(AIことはじめ)」という名称は、江戸時代の蘭方医・杉田玄白が著した『蘭学事始』にちなんでいます。師となる人がほとんどいない中、手探りでオランダ語の解剖書『ターヘル・アナトミア』を読み解いていった蘭学者たちの姿を、生成AIという新しい道具に独学で向き合う現代の医療者に重ねました。

## 設計原則

本サイトは、以下の4つの原則に沿って作られています。

1. **教員の手間を増やさない。** 教員の仕事は「ここを見ておいて」とリンクを送ることだけです。
2. 学習者は体験、フィードバック、目録の発行まで、誰かの手を借りずに一人で完結できます。
3. 保守は運営者一人がMarkdown編集だけで回ります。AIへの指示文の正本もリポジトリ内のMarkdownで管理しています。
4. **ユーザーのログ・個人情報をサイト側に一切残しません。**(クイズ合格状態のlocalStorage保存は端末内で完結するため、この原則に抵触しません)

## 誤りの報告・改善要望 { #report-issues }

本サイトはオープンソースで開発しており、内容の誤り(誤字、リンク切れ、記述の不正確さなど)の報告や改善の要望を、[GitHubリポジトリのIssue](https://github.com/youkiti/ai-kotohajime/issues)で受け付けています。次の流れで投稿してください。

1. **既存のIssueを確認する** — [Issue一覧](https://github.com/youkiti/ai-kotohajime/issues)を開き、同じ内容がすでに報告されていないか確認します(閲覧にアカウントは不要です)。
2. **新しいIssueを作成する** — [新規Issue作成ページ](https://github.com/youkiti/ai-kotohajime/issues/new)でタイトルと本文を記入し、投稿します(投稿には無料のGitHubアカウントが必要です)。
3. **本文に次の情報を含める**:
    - 対象ページのURL(またはユニット名・ページ名)
    - 気づいた内容(誤字/リンク切れ/内容の誤り/表示崩れ/改善案 など)
    - 表示の不具合の場合は、お使いのブラウザと端末の種類

!!! warning "Issueは誰でも閲覧できる公開の場です"
    患者情報を含む個人情報や、公開できない情報はIssueに書かないでください。

GitHubアカウントをお持ちでない方向けの連絡先は準備中です(運営名義の確定にあわせて整備します)。
